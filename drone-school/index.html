Program 01

Drone Programming
with DJI Tello.

From first flight commands to autonomous drone missions.

This pathway introduces students to drones through safe indoor flight, visual programming, Python, mission planning, and autonomous drone projects using DJI Tello drones. DJI Tello is best suited to indoor learning — the easiest entry point for young learners and beginners.

Level 1Drone Basics & Visual Programming

Students learn drone safety, basic flight logic, movement commands, and simple visual programming.

  • What is a drone?
  • Drone safety
  • Takeoff and landing
  • Basic movement
  • Visual / Scratch-style programming
  • Simple command sequences

Outcome: students can safely control a drone and create simple programmed movements.

Level 2Python Drone Missions

Students move from visual logic to Python and learn how to create planned drone missions.

  • Python basics
  • Drone control with code
  • Variables, loops, conditions, functions
  • Mission planning
  • Debugging drone behavior

Outcome: students can write Python code to control drone missions.

Level 3Autonomous Drone Projects

Students create more advanced drone projects using mission logic, camera concepts, and autonomous behavior.

  • Advanced mission logic
  • Multi-step missions
  • Camera and video-stream basics
  • Introduction to computer vision
  • Object tracking concepts
  • Final autonomous project
  • Swarm / multi-drone concepts when equipment allows

Outcome: students can create a small autonomous drone project.

Program 02

Drone Building
with FabLab.

Learn how drones are designed, manufactured, assembled, and tested.

A practical engineering course — one complete pathway, not multiple levels. Students learn how drone components work together and build a real drone using FabLab tools and electronics.

What you'll learn
  • Drone components
  • Frame and chassis design
  • CAD basics
  • Materials and manufacturing
  • 3D printing and fabrication
  • Motors and propellers
  • ESCs, batteries, and power systems
  • Flight controller basics
  • Wiring and assembly
  • Calibration basics
  • Testing and troubleshooting
  • Safety checks
Built for
  • Students interested in engineering
  • Makers
  • University students and technical-school students
  • Engineering faculties
  • Young people drawn to manufacturing and robotics

Outcome: students understand how drones are built and gain hands-on engineering experience by assembling and testing a drone.

Program 03

Official Drone
Pilot Training.

For adults preparing for legal and professional drone operation.

Designed for adults who want to operate drones professionally and understand drone regulations, safety, mission planning, and practical flight. Curriculum aligns with the regulatory framework being established for civil drone operation in Uzbekistan.

Training topics
  • Drone regulations
  • Airspace awareness
  • Flight safety
  • Operator responsibility
  • Pre-flight checklist
  • Practical flight training
  • Emergency procedures
  • Mission planning
  • Weather and environment awareness
  • Risk assessment
  • Final assessment process
Built for
  • Adults aged 18+
  • Future drone pilots and operators
  • Professionals and entrepreneurs
  • Company employees needing certified pilots
  • Organisations operating drones at scale

Outcome: learners gain the knowledge and practical skills needed for safe and professional drone operation.
