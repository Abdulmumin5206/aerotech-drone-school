# Aerotech Drone School

Uzbekistan's first drone programming school website. A modern, responsive landing page featuring hands-on courses where students learn to code, build, and pilot autonomous aircraft.

## Features

- Full-screen video hero section
- Responsive design
- Modern UI with custom animations
- Program showcase
- Interactive accordions
- Course information

## Technologies Used

- HTML5
- CSS3 (Custom Properties, Grid, Flexbox)
- Vanilla JavaScript
- SVG Graphics

## Live Demo

Visit the live site: [Aerotech Drone School](https://abdulmumin5206.github.io/aerotech-drone-school/)

## Local Development

1. Clone the repository
2. Open `index.html` in your browser
3. No build process required!

## Project Structure

```
├── index.html          # Main HTML file
├── css/
│   ├── main.css       # Main stylesheet
│   ├── variables.css  # CSS custom properties
│   ├── base.css       # Base styles
│   ├── layout.css     # Layout styles
│   ├── components.css # Component styles
│   └── sections.css   # Section-specific styles
├── js/
│   └── main.js        # JavaScript functionality
└── assets/
    └── videos/        # Video assets
```
